## Featured Projects

### [LLM Evaluation & Prompt Engineering Portfolio](projects/llm-evaluation-portfolio/README.md)

A collection of prompt experiments, model failure analysis, and evaluation frameworks designed to improve large language model performance.

Topics include:

- Prompt optimization
- Reasoning evaluation
- Hallucination detection
- Alignment testing
- Benchmark design

---

### [LLM Failure Analysis Dataset](projects/llm-failure-analysis/README.md)

A research-style dataset documenting common failure patterns in language models including:

- Reasoning errors
- Hallucinated citations
- Instruction-following failures
- Prompt injection vulnerabilities

---

### [Prompt Testing Framework](projects/prompt-testing-framework/README.md)

A small toolkit demonstrating how prompts can be programmatically tested and evaluated using structured prompts and evaluation metrics.
